Media Library Configuration

Enter the name of the directory within the application folder to use as your media library (e.g., 'videos', 'media_collection'). This directory must be located directly inside the application's main folder: {{ CWD_DISPLAY_PATH }}. Special characters and paths (slashes) are not allowed.

Choose how many media items to display per row in the grid. Small shows more items, Large shows fewer but bigger items.

When enabled, videos will automatically start playing when you visit the video player page. Most browsers require videos to be muted for autoplay to work.

When muted, videos will start without sound and show an unmute button. When unmuted, videos will start with sound (but autoplay may not work in some browsers).

When enabled, the next video in the queue will automatically start playing when the current video ends. This creates a continuous viewing experience.

When enabled, the current video will automatically restart from the beginning when it reaches the end. Takes priority over autoplay next.
